생생한 향수 정보 제공 어플 HMOA

## 🛠️ 개발 환경
- Xcode 15.3
- Swift 5.10
- iOS 15.0 이상

  

## 📁 프로젝트 디렉토리 구조
<div style="display: flex; align-items: flex-start;">
  <img src="https://github.com/HMOAA/HMOA_iOS/assets/75370733/f54bcb03-3b07-497c-a8cf-c9523cb84c6b" alt="Project Structure" width="300">
  <ul>
    <li><strong>Application:</strong> 앱의 진입점과 초기 설정 관련 코드(AppDelegate, SceneDelegate)</li>
    <li><strong>Manager:</strong> 키체인, 로그인 등 다양한 매니저 클래스</li>
    <li><strong>Network:</strong> API 호출과 관련된 네트워크 레이어</li>
    <li><strong>Presentation:</strong> UI 레이어와 관련된 뷰, 뷰모델, 뷰컨트롤러</li>
    <li><strong>Extensions:</strong> 확장 메서드 및 유틸리티</li>
    <li><strong>Resource:</strong> 리소스 파일들</li>
  </ul>
</div>
